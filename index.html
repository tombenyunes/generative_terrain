Procedural Terrain Generator


Everything you see is procedurally generated using Improved Perlin Noise and rendered with Three.js and OpenGL .

To generate new terrain click Regenerate at the top of the options panel.

To Orbit use Left Mouse Button.
To Pan use Right Mouse Button.
To Zoom use Scroll Wheel.

Terrain seeds can be saved and reused, and the entire scene can be downloaded as a group of 3D models.

Width/Depth = Defines the dimensions of the terrain generated.
Terrain Formation = Changes the generation algorithm to replicate different types of land formation/biomes.
Water Type = Basic water uses a simple cubemap reflection, whereas Advanced water uses a vertex shader to simulate ripples and generate real-time reflections (GPU-intensive).
Height/Noise Distribution = Changes how the noise output is vertically and horizontally distributed. Can be used to create very different terrain structures.
Block/Light Colours = Allows the colour of each block type and both lights to be changed to any hex colour.
